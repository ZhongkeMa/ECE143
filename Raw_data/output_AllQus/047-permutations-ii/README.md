# [Permutations II][title]

## Description

Given a collection of numbers that might contain duplicates, return all
possible unique permutations.

**Example:**
            Input: [1,1,2]    Output:    [      [1,1,2],      [1,2,1],      [2,1,1]    ]    


**Tags:** Backtracking

**Difficulty:** Medium
