# [Find Largest Value in Each Tree Row][title]

## Description

You need to find the largest value in each row of a binary tree.

**Example:**  
            Input:                   1             / \            3   2           / \   \            5   3   9         Output: [1, 3, 9]    


**Tags:** Tree, Depth-first Search, Breadth-first Search

**Difficulty:** Medium
