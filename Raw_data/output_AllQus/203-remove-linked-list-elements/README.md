# [Remove Linked List Elements][title]

## Description

Remove all elements from a linked list of integers that have value **_val_**.

**Example:**
            Input:  1- >2->6->3->4->5->6, _**val**_ = 6    Output: 1- >2->3->4->5    


**Tags:** Linked List

**Difficulty:** Easy
