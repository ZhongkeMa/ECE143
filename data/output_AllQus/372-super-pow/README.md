# [Super Pow][title]

## Description

Your task is to calculate _a_ _b_ mod 1337 where _a_ is a positive integer and
_b_ is an extremely large positive integer given in the form of an array.

**Example 1:**
            Input: a = 2, b = [3]    Output: 8    

**Example 2:**
            Input: a = 2, b = [1,0]    Output: 1024

**Difficulty:** Medium
