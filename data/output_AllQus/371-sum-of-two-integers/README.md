# [Sum of Two Integers][title]

## Description

Calculate the sum of two integers _a_ and _b_ , but you are **not allowed** to
use the operator `+` and `-`.

**Example 1:**
            Input: a = 1, b = 2    Output: 3    

**Example 2:**
            Input: a = -2, b = 3    Output: 1    


**Tags:** Bit Manipulation

**Difficulty:** Easy
