# [Palindrome Partitioning][title]

## Description

Given a string _s_ , partition _s_ such that every substring of the partition
is a palindrome.

Return all possible palindrome partitioning of _s_.

**Example:**
            Input:  "aab"    Output:    [      [ "aa","b"],      ["a","a","b"]    ]
