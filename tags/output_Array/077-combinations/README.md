# [Combinations][title]

## Description

Given two integers _n_ and _k_ , return all possible combinations of _k_
numbers out of 1 ... _n_.

**Example:**
            Input:  n = 4, k = 2    Output:    [      [2,4],      [3,4],      [2,3],      [1,2],      [1,3],      [1,4],    ]    


**Tags:** Backtracking

**Difficulty:** Medium
